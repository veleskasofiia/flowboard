Technology

Tech Stack & Why

Every technology was chosen for a specific reason — here is what powers FlowBoard.

▲

Next.js 16 (App Router)

Frontend + Backend
Why

Full-stack React framework. API routes live alongside pages — no separate backend needed. Server components keep secrets out of the browser.

🔐

Supabase

Auth + Database
Why

Authentication and Postgres database out of the box. Stores user accounts, chat history, and saved workflow canvases. Generous free tier.

🤖

Groq — LLaMA 3.3 70B

AI / LLM
Why

Powers the AI chat assistant. Groq runs LLaMA 3.3 at extremely high speed — perfect for real-time chat. Free developer tier.

🔌

Composio

Integrations + Tool Use
Why

Handles OAuth for Gmail, Slack, Notion, Google Calendar and more via one API. Also lets the AI execute real actions on behalf of users.

🔵

ReactFlow

Visual Canvas
Why

Drag-and-drop canvas for building workflows visually. Provides nodes, edges, zoom/pan, and custom rendering. Canvas auto-saves to localStorage.

🎞

Framer Motion

Animations
Why

Smooth entrance animations on dashboard tiles and cards. Keeps the UI polished without writing custom CSS keyframes.

🔷

TypeScript

Type Safety
Why

Strict typing across all components and API routes. Catches shape mismatches at compile time rather than at runtime.

☁️

Vercel

Deployment
Why

Zero-config deployment for Next.js. Push to GitHub → auto builds and deploys. CDN, env vars, and preview deployments — free on Hobby plan.

Architecture

How It Works

Full request lifecycle — from the browser all the way to third-party APIs.

🌐 Browser — Next.js React

User logs in, builds a canvas, sends a chat message

↓

⚙️ Next.js API Routes (server-side)

/api/agent — AI chat  ·  /api/connect — OAuth  ·  /api/summary — live app data

↓ splits into ↓

🤖 Groq API

LLaMA 3.3 70B. Receives the message, decides which Composio tool to call, returns the final reply.

🔐 Supabase

Verifies auth session, stores chat messages and saved canvases.

↓

🔌 Composio

Executes real actions on Gmail, Slack, Notion, Google Calendar using the user's OAuth token

↓

📬 Third-party APIs

Gmail API · Slack API · Notion API · Google Calendar API — real data returned to the user
